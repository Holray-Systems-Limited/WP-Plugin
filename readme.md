# Holray WordPress Plugin
This is the repository for the Holray WordPress Plugin. Please download the latest version from here: https://github.com/Holray-Systems-Limited/WP-Plugin/releases

## Installation
Simply download the latest ZIP release from the link above and install into your WordPress installation.

You can do this, by accesing the admin area of WordPress, then Plugins, "Add Plugin" and simply upload your ZIP file.
