Holray WordPress Plugin Upgrade Guide
=====================================

Version 1.2.0
-------

This version has one breaking change that requires manual review when updating. We estimate that upgrading to 1.2.0 will take around 10 minutes.

If you haven't overwritten any of our template files, there is nothing for you to do!

* * *

Holray Unit Pages
------------

If you have updated the `single-holray_unit.php` template file, you will need to follow the steps below benefits of having unit calendars on your unit pages.

You will see that in [our template file we have added the following code](https://github.com/Holray-Systems-Limited/WP-Plugin/blob/f691c590d860205dc6562045503a21344f6dabb6/templates/single-holray_unit.php#L84). This adds the content written in Gutenberg / WordPress Classic Editor onto the page, as well as, adds a column for the unit calendar.

To add the calendar into your version of our template, you will need to add the following code into this file:
```php
<div 
    id="holray-unit-calendar" 
    class="holray-calendar" 
    data-holray-url="<?php echo esc_url(\Holray\Plugin\Util\Url::getCustBookUrl()); ?>" 
    data-holray-cal-url="<?php echo esc_url(admin_url("admin-ajax.php")); ?>" 
    data-holray-unit="<?php echo esc_attr(get_post_meta(get_the_ID(), "holray_class", true)); ?>"
>
    <div class="holray-loader"><div></div></div>
</div>
```

You can update the CSS as you see fit for calendars across the site, all classes are prefixed with `holray-calendar`. To give a good starting place you can copy our [CSS from here.](https://github.com/Holray-Systems-Limited/WP-Plugin/blob/f691c590d860205dc6562045503a21344f6dabb6/src/css/holray.css#L553)
